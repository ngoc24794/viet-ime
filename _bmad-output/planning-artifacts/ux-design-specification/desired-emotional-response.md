# Desired Emotional Response

### Primary Emotional Goals

**Core Emotional Value:** "Finally, I can type Vietnamese anywhere"

**Primary Feeling: Accomplishment and Relief**
   - User achieves what was previously impossible (Vietnamese in terminals/IDEs)
   - Relief from no longer struggling with Unikey limitations
   - Sense of capability: "I can do what I need"

**Secondary Feelings:**
- **Confidence:** Always know which mode is active, no guessing
- **Trust:** Reliable behavior that just works, every time
- **Efficiency:** Zero friction, effortless mode switching
- **Delight:** Exceeds expectations, smoother than expected
- **Productivity:** "No more copying from external sources"

**Emotions to Avoid:**
- **Frustration:** Unclear mode indication, stuck in wrong mode
- **Confusion:** "Does it work in this app?"
- **Anxiety:** Typing lag, broken output
- **Disappointment:** "Just like Unikey, doesn't work in terminal"
- **Skepticism:** "Maybe this will work" (after first failure)

### Micro-Emotions

**Confidence vs. Confusion (Most Critical)**
- **Design Implication:** Crystal-clear visual mode indication in system tray
- **Implementation:** Instant visual feedback on mode switch (Ctrl+Shift)
- **Goal:** No guessing - user always knows if VI or EN mode is active
- **Success Criteria:** Glance at system tray = instantly know current mode

**Trust vs. Skepticism (Most Critical)**
- **Design Implication:** Consistent, reliable behavior across all applications
- **Implementation:** Clipboard-based output must work flawlessly in terminals
- **Goal:** Build trust through reliability, not promises
- **Success Criteria:** "I know it will work" - user confidence before action

**Accomplishment vs. Frustration (Most Critical)**
- **Design Implication:** First terminal success must feel magical
- **Implementation:** Celebrate first successful Vietnamese output in terminal
- **Goal:** User feels proud and capable
- **Success Criteria:** Vietnamese appears correctly where Unikey never worked
